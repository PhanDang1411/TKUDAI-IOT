3. Metadata ảnh

Metadata cho biết ảnh đến từ đâu, khi nào, kích thước bao nhiêu, độ sáng và đường dẫn lưu ở đâu.

4. Event ảnh

Event là thông tin vận hành sinh từ ảnh. Với motion, người và động vật được ghi thành event riêng: PERSON_DETECTED hoặc ANIMAL_DETECTED.
